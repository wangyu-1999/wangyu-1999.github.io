命名规范往往和小组的约定有关，有的习惯用驼峰来命名，有的习惯用下划线来命名，但有一些共同的良好实践，比如变量采用名词短语、函数采用动词短语等等.....

首先按照词性总结一下命名的常用词。

## 泛指名词

- count
- title
- name
- user
- account
- list
- option
- map
- node
- group
- label
- field
- value
- id
- children
- index
- page

- form
